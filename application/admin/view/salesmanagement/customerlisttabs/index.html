客户列表
  1. 新客户：数据来源由推广分配给各部门内勤 ，再由各部门内勤分配给具体的销售，在新客户列表中也可以添加自己的客户渠道。

  2. 待联系：以下几种客户跟进状态可以归纳等级为待联系（如：客户电话无法接通、忙、占线、关机等。。。），仅供参考，具体以沟通内容/结果酌情判断客户等级。

  3. 有意向：以下几种客户跟进状态可以归纳等级为有意向（客户有考虑购/租车的意向、约定时间可以到店看车等。。。），仅供参考，具体以沟通内容/结果酌情判断客户等级。

  4. 暂无意向：以下几种客户跟进状态可以归纳等级为暂无意向（客户目前/短期时间内不会考虑购/租车、持观望状态等。。。），仅供参考，具体以沟通内容/结果酌情判断客户等级。

  5. 已放弃：销售员已放弃的客户。

  6. 跟进时间过期客户： 系统自动判断 当前时间大于销售员反馈的下次跟进时间，如果大于这个时间，那么就存在未及时跟进客户，需要尽快与客户取得联系。

  7. 针对58同城的客户： 需要尽快与客户取得联系，获取客户的其他联系方式。

  8. 关于客户姓名前的字母： 姓名前“S”表示意向是私家车，“U”表示意向是新能源货车，无字母表示意向网约车
